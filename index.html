Dinam

Software de gestión para centros médicos v1.0

View on GitHub Download .zip Download .tar.gz

¿Para qué empresas u organizaciones sirve?.

El sistema de reserva de horas, sirve para cualquier tipo de empresa o institución que desee gestionar horas (ya sea para médicos, profesionales, inmuebles, etc...) y tener un control con alertas vía e-mail o por la misma plataforma

Plataforma Saas

Dinam es un servicio que se puede ofrecer como Saas (Software como servicio) pudiendo ser arrendado por clientes o ser instalado de manera local, es un software libre lo puedes utilizar como quiera.

¿Es gratis y de licencia libre?

Si, Es un proyecto libre y por todas partes es gratis, la razón de esta decisión es para contribuir a un mundo con mejores sistemas informáticos.

Tecnologías involucradas

El desarrollo está basado en PHP y MySQl utilizando el framework Laravel 5.2.

Requisitos del servidor

  • PHP >= 5.5.9
  • MySql
  • Nginx o Apache Server
  • Composer

Instalación

  • Descargar el proyecto y crear un virtualhost en apache o una nueva configuración de servidor en nginx
  • Descargar las librerías necesarias ejecutando "composer install" desde la raíz del proyecto
  • Configurar el archivo .env para la conexión de la base de datos (debe crear una base de datos en MySql)
  • Ejecutar el comando "php artisan migrate", para generar las entidades de la base de datos de manera automática
